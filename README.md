## รายละเอียดโปรแกรม
โปรแกรมนี้เป็นระบบคำนวณเกรดของนักศึกษาในรายวิชา โดยพัฒนาด้วยภาษา C# และใช้แนวคิด Object-Oriented Programming (OOP)

## ความสามารถของระบบ
- เพิ่มรายวิชา (ชื่อวิชา + รหัสวิชา)
- เพิ่มนักศึกษา (ชื่อ, รหัส, คะแนน)
- คำนวณเกรดอัตโนมัติ
- แสดงผล:
  - รายชื่อนักศึกษาทั้งหมด
  - คะแนนสูงสุด / ต่ำสุด
  - นักศึกษาที่ไม่ผ่าน (F)
  - ค่าเฉลี่ยของคะแนน
- มีเมนูให้เลือกใช้งาน และสามารถออกจากโปรแกรมได้

### 2. Inheritance

### 3. Polymorphism

### 4. Abstraction
- ใช้ abstract class `Person`
